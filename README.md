## Project Overview:
The Bank Loan Analysis Project focuses on analyzing loan application data to evaluate lending performance, borrower behavior, and loan risk. The project helps financial institutions make data-driven decisions by tracking key loan metrics such as approvals, repayments, defaults, and overall portfolio health through interactive dashboards.
## Dataset Used:
Bank loan application dataset containing:
Loan details (loan amount, interest rate, term)
Customer demographics (employment length, income, home ownership)
Loan status (Fully Paid, Charged Off, Current)
Issue date, purpose, grade, and verification status
Data was cleaned and transformed to ensure accuracy and consistency before analysis.
